## STRZAŁY / STRZAŁY CELNE
1. 🟢 ⭐⭐ Faworyt vs **głęboki blok** rywala → więcej strzałów, ale mniej celnych i więcej zablokowanych.
2. 🟡 ⭐⭐ Napastnik vs **wolna/słaba para stoperów** → strzały ↑ (dane: pojedynki przegrywane przez obrońców rywala).
3. 🟢 ⭐⭐ **Wykonawca stałych fragmentów / karnych** → strzały i celne ↑ (dane: flaga egzekutora — do ustawienia).
4. 🟡 ⭐⭐ Skrzydłowy **wchodzący do środka** vs boczny obrońca, którego rzadko odbierają → strzały ↑.

## STRZAŁY ZZA POLA KARNEGO
5. 🟢 ⭐⭐⭐ **Strzelec z dystansu** vs głęboki blok (brak miejsca w polu) → zza pola ↑.
6. 🟡 ⭐⭐ Rywal **odcina podania w pole karne** → wymusza strzały z dystansu.

## STRZAŁY ZABLOKOWANE
7. 🟡 ⭐⭐ Strzelec vs drużyna z **dużą liczbą bloków** (dużo ciał w polu) → zablokowane ↑.
8. ✅ ⭐⭐ Powiązanie: strzały zza pola → automatycznie więcej zablokowanych.

## STRZAŁY GŁOWĄ / CELNE GŁOWĄ
9. 🟡 ⭐⭐⭐ **Target man** vs niski/słaby w powietrzu stoper → głową ↑ (dane: pojedynki powietrzne).
10. 🟡 ⭐⭐⭐ Drużyna **dużo dośrodkowująca** × wzrostowy napastnik → głową ↑ (dane: dośrodkowania).
11. 🟢 ⭐⭐ Dużo **rzutów rożnych** w meczu × wzrostowi zawodnicy → głową ↑ (dane: rożne — mamy).

## STRZAŁY W 1. POŁOWIE
12. 🟢 ⭐⭐ **Szybko zaczynająca** drużyna vs wolno rozkręcający się rywal → 1P ↑ (dane: rozkład strzałów na połowy — mamy).
13. 🟢 ⭐ Drużyna, która **broni wyniku od przerwy** → koncentracja strzałów w 1P.

## FAULE POPEŁNIONE
14. ✅ ⭐⭐⭐ Obrońca vs **drybler** rywala → faule ↑.
15. ✅ ⭐⭐ Rywal **fizyczny** (dużo pojedynków) → faule ↑.
16. 🟡 ⭐⭐⭐ Obrońca **często ogrywany 1v1** vs dobry drybler → faule ↑ (dane: „dribbled past" / przegrane pojedynki).
17. 🟡 ⭐⭐ Rywal grający **szybkie kontry** → faule taktyczne ↑ (dane: tempo/kontry rywala).
18. 🟢 ⭐⭐ **Defensywny pomocnik** vs drużyna grająca środkiem → faule ↑.

## FAULE WYWALCZONE
19. ✅ ⭐⭐⭐ **Drybler** vs faulujący rywal → wywalczone ↑.
20. 🟡 ⭐⭐ **Holdup striker** (gra ciałem, tyłem do bramki) vs fizyczny stoper → wywalczone ↑ (dane: dotknięcia w tercji + pojedynki).
21. 🟢 ⭐⭐ Rozgrywający w **zatłoczonym środku** → faulowany, by zatrzymać akcję.
22. 🟡 ⭐⭐ **Szybki** zawodnik wbiegający za linię → ścinany → wywalczone ↑ (dane: proxy szybkości).

## ODBIORY
23. ✅ ⭐⭐⭐ Obrońca/pomocnik vs **drybler** → odbiory ↑.
24. ✅ ⭐⭐ Rywal **fizyczny** → więcej starć → odbiory ↑.
25. 🟡 ⭐⭐⭐ **Boczny obrońca vs skrzydłowy PO JEGO STRONIE** (świadomość lewa/prawa) → odbiory ↑ (dane: pozycja L/P).
26. ✅ ⭐⭐ **Underdog** broniący głęboko → więcej obrony → odbiory ↑ (przez scenariusz meczu).

## PRZECHWYTY
27. ✅ ⭐⭐ Pomocnik vs **ryzykownie grający** (dryblujący) rywal → przechwyty ↑.
28. 🟡 ⭐⭐⭐ Obrońca vs **drużyna grająca długimi piłkami** (direct) → przechwyty ↑ (dane: długie podania rywala).
29. 🟢 ⭐⭐ Głęboki obrońca vs rywal z **wysokim posiadaniem** → więcej okazji do przechwytu (dane: posiadanie — mamy).

## SPALONE
30. ✅ ⭐⭐⭐ Napastnik vs **wysoka linia** (pułapki ofsajdowe) → spalone ↑.
31. 🟢 ⭐⭐ **Poacher grający na ramieniu** obrońcy (historia spalonych) vs agresywna linia → efekt spotęgowany.
32. 🟡 ⭐⭐ Drużyna z **dużą liczbą podań za linię** → spalone ↑ (dane: through-balls).

## ŻÓŁTE KARTKI
33. ✅ ⭐⭐ Obrońca vs drybler → kartka ↑ (przez faule).
34. 🟢 ⭐⭐⭐ Faulujący zawodnik × **surowy sędzia** × derby (mamy sędziego i flagę rangi meczu).
35. 🟡 ⭐⭐ **Boczny obrońca vs gwiazda-skrzydłowy** → cyniczne faule → kartka ↑ (dane: pozycja L/P).
36. 🟡 ⭐⭐ **Ostatni obrońca / wysoka linia** vs szybki napastnik → faule w groźnych strefach → kartka ↑.

## RYNKI DRUŻYNOWE (faule / kartki / strzały)
37. 🟢 ⭐⭐⭐ Faule drużyny vs rywal z **dryblującymi** wywalczającymi faule → team_fouls ↑.
38. 🟡 ⭐⭐ **Wysoki pressing** drużyny → własne faule ↑ (dane: PPDA / odzyski na połowie rywala).
39. ✅ ⭐⭐ Underdog → team_fouls ↑ (scenariusz meczu).
40. 🟢 ⭐⭐ **Dwie agresywne drużyny** naraz → kartki drużyny ↑.

## Mechanizmy przekrojowe (upgrade całej warstwy, nie pojedyncza analogia)

- **A. Świadomość strony L/P** 🟡 ⭐⭐⭐ — dopasuj bocznego obrońcę do skrzydłowego rywala po tej samej stronie. Największy skok dokładności dla odbiorów/fauli/kartek. Wymaga rozpoznania pozycji lewa/prawa.
- **B. Granularność ról** 🟡 ⭐⭐ — rozróżniaj boczny obrońca / stoper / DM / „10" / skrzydło / napastnik, nie tylko O/P/N. Wyostrza wszystkie analogie.
- **C. Słabość „ogrywany 1v1"** 🟡 ⭐⭐⭐ — obrońca często przechodzony → więcej fauli/kartek vs dryblerzy. Wymaga pola „dribbled past".
- **D. Model stałych fragmentów** 🟡 ⭐⭐ — rożne + dośrodkowania × gra głową → strzały głową i rynki drużynowe.
- **E. Model głębokiego bloku** 🟡 ⭐⭐⭐ — rywal broniący nisko → zza pola ↑, zablokowane ↑, celne ↓. Spina rynki strzałów.
